# Setup JDK
Create a `productid.jdk` containing `/home/home/opt/jbr-graalvm/current` corresponding to each `~/.config/JetBrains/Product.Version/productid.key`
# Setup remote dev
Run `bin/remote-dev-server.sh registerBackendLocationForGateway` for each IDE under `~/.local/share/JetBrains/Toolbox/apps`
# Add the following into each .vmoptions
-Xms4096m
-Xmx4096m
-XX:+AlwaysPreTouch
-XX:+UseTransparentHugePages
-Dsun.java2d.uiScale=2
-Dide.browser.jcef.skip_module_check=true
